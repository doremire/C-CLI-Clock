## Features

A graphical clock written in ASCII art.

- Lightweight system written in low-code C language.

## Installation

```
sudo make install
```

## Usage

```

    cli-clock -c 5 -t "🪐 Good night"

    ----------------------------------------------------------------
    Options:

    -h Help
    -c [1-7] Change colors
    -t "text" Add text

```
